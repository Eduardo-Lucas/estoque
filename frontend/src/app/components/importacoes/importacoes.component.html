Categoria e fornecedor são identificados pelo nome; se não existirem, são criados automaticamente.

Se já existir um produto com o mesmo nome, os campos presentes na linha são atualizados (células vazias não sobrescrevem valores já cadastrados); caso contrário, o produto é criado.

Reimportar o mesmo arquivo é seguro: itens já processados não duplicam estoque.
